# 🌟 MetaLife OS - 통합 AI 생산성 플랫폼

**완전한 엔드-투-엔드 AI 기반 생산성 및 자동화 플랫폼** - 콘텐츠 생성, 웹 개발, 시스템 자동화, 교육을 하나로 통합합니다.

## 🎯 비전

MetaLife OS는 개인과 팀이 AI 기술을 활용하여 다음을 수행할 수 있는 통합 플랫폼입니다:

- 🤖 **AI 에이전트 시스템**: Mother-Child AI 아키텍처
- 📝 **콘텐츠 자동화**: 7단계 완전 자동화 파이프라인
- 🌐 **웹 개발**: 포트폴리오 사이트 자동 생성 및 관리
- 🔧 **시스템 자동화**: 개발 워크플로우, 배포, 테스팅
- 🎓 **교육 플랫폼**: AI 기반 개인화된 학습 경험
- 🎨 **미디어 생성**: 이미지, 프레젠테이션 자동 생성

## 🚀 Quick Start

### 설치

```bash
# 레포지토리 클론
git clone https://github.com/ProCodeJH/MetaLife_OS.git
cd MetaLife_OS

# 개발 환경 시작
npm run docker:dev

# 접속
- 웹 앱: http://localhost:3000
- AI 에이전트: http://localhost:8000
- 모니터링: http://localhost:3001
```

### 환경 설정

```bash
cp .env.example .env
# .env 파일에 API 키들 설정
```

## 🏗️ 핵심 기능

### 🤖 Mother-Child AI 아키텍처
- Mother AI: 헌법 기반 통제 시스템 (실행/승인 권한 독점)
- 60+ Child AI: 권한/기억 없는 부품형 지능 (오직 사고만)
- 안전 제한: 벤치마크 기반 증명
- 재현성 보장: 동일 입력에서 95% 일관된 결과

### 📝 콘텐츠 자동화 엔진
1. **INGEST**: SHA256 기반 중복 제거 및 메타데이터 추출
2. **TRANSCRIBE**: OpenAI Whisper로 고품질 자막 생성
3. **GENERATE**: GPT-4로 플랫폼별 최적화된 콘텐츠 생성
4. **VALIDATE**: 5차원 품질 검증 (Hook, 관련성, 가독성, SEO, 독창성)
5. **RENDER**: 하이라이트 기반 Shorts 자동 생성
6. **PUBLISH**: WordPress, YouTube, Naver Blog 자동 발행
7. **MEASURE**: 성과 분석 및 ROI 측정

### 🌐 웹 스튜디오
- AI 기반 웹사이트 자동 생성
- 실시간 편집 및 미리보기
- 반응형 디자인
- 자동 배포 (Vercel, Netlify)

### 🎓 교육 플랫폼
- AI 기반 개인화된 학습 경로
- 게이미피케이션 시스템
- 인터랙티브 콘텐츠
- 진도 추적 및 성과 분석

## 🛠️ 기술 스택

### 프론트엔드
- **Next.js 14**: React 풀스택 프레임워크
- **TypeScript**: 타입 안전성
- **Tailwind CSS**: 모던 CSS 프레임워크

- **PostgreSQL**: 메인 데이터베이스
